# ☕ PalamiGoPOS

**PalamiGoPOS** is an offline-first Android Point of Sale (POS) application designed for small coffee shop businesses.  
Built using **Kotlin + XML in Android Studio**, this app helps manage orders, payments, inventory, and daily reports without needing internet.

---

## 📱 Features

### 🛒 POS System
- Add products to cart
- Automatic quantity handling
- Real-time total calculation
- Fast and simple cashier workflow

### 💵 Checkout
- Cash input system
- Automatic change calculation
- Exact payment detection
- Prevents insufficient payment
- Saves transactions offline

### 📦 Inventory Management
- Add new products
- Edit product details
- Delete products
- Toggle ON/OFF product visibility in POS
- Category-based organization

### 📊 Reports
- Daily sales summary
- Total sales per day
- Total number of orders per day
- Organized by date (each day = separate row)

### 📜 Order History
- View all transactions
- Detailed order view
- Shows date, total, payment, and change

### 📱 Responsive Design
- Works on phones and tablets
- Supports portrait and landscape mode
- Adaptive product grid layout
- Clean and user-friendly UI

---

## 🛠️ Tech Stack

- **Language:** Kotlin
- **UI:** XML Layouts
- **Architecture:** MVVM
- **Database:** Room (Offline)
- **Components:**
  - RecyclerView
  - ViewBinding
  - Material Design

---

## ⚙️ Requirements

- Android Studio (latest recommended)
- Minimum SDK: **24 (Android 7.0)**
- Target SDK: **36**
- Device: Android Phone or Tablet
